## 最小心智模型

```text
Idea → Tiny MVP → Day 1 command → Acceptance Gate → Next action
```

默认模式已经预配置好：

- Compact output first
- 1-3 MVP features
- 2-4 tools
- One primary agent first
- Advanced playbooks hidden until needed

---

## Do Not

- Do not read every playbook before starting.
- Do not switch tools before Day 1 works.
- Do not add a fourth MVP feature.
