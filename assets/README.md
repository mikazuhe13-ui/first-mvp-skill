# Assets

## Workflow

Put visual assets here so the README can stay clean and maintainable.

---

## Purpose

Store the FirstMVPSkill logo, screenshots, and demo images.

---

## Recommended files

```text
assets/logo.png
assets/demo-output.png
assets/quick-start.gif
```

After adding `assets/logo.png`, place it near the top of `README.md`:

```md
<p align="center">
  <img src="assets/logo.png" alt="FirstMVPSkill logo" width="160">
</p>
```

---

## Do Not

- Do not add large raw design files.
- Do not use broken image links in the README.
- Do not commit local design drafts unless they are meant for release.
