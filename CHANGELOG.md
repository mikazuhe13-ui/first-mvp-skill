# Changelog

All notable changes to FirstMVPSkill will be documented in this file.

---

## [1.0.0] - 2026-05-31

### Added
- Core skills: `first-mvp-launch` and `agent-command-coach`
- 6 command templates: planning, coding, review, anti-overbuilding, multi-agent dispatch, execution feedback loop
- 5 templates: intake form, MVP plan, 7-day plan, agent task brief, project status
- 6 checklists: MVP scope, agent command, context management, anti-tool-hopping, acceptance gate, launch readiness
- 5 examples: AI study assistant, personal knowledge base, student competition demo, AI content tool, closed-loop full example
- 13 Agent Operation Playbooks: agent mode matrix, tool mode selector, tool-specific workflows, reasoning effort, subagent, long task, human review gates, current agent adapter, capability checklist, fallback patterns, slash commands
- Routing layer: skill router, compact output rules, token budget policy
- Docs: installation, quick start, guide, troubleshooting, FAQ, contributing, API reference, commit convention, writing standards
- START_HERE.md: bilingual entry point for new users
- config/default-mode.md: default Compact Mode behavior
- Bilingual README (Chinese + English)
- Mermaid flowchart for core loop
- Hero image and visual assets
- Live demo: AI Action Plan Generator (HTML, zero dependencies)
- GitHub issue and PR templates
- Agent Skill Ladder (Level 0-5 progression guide)

### Changed
- Positioning broadened from "beginner-first" to "decision-making system" — now also serves users who can't choose between tools, modes, or configurations
- README hero tagline now leads with "Can't choose a tool? Can't pick a mode?"
- "Choose your path" table expanded with tool/mode selection entries
- "How this differs" table now includes "Tool comparisons" row
- "Why this exists" section updated to emphasize anti-decision-fatigue
